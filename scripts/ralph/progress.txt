## Codebase Patterns
- Always run `pnpm build` before `pnpm migrate:generate` - drizzle-kit reads from dist/ not src/
- For idempotent MySQL migrations, use information_schema.TABLE_CONSTRAINTS to check existence
- Drizzle migrations are in apps/server/drizzle/ with meta files in drizzle/meta/
- All Echoe relation fields use varchar(191) for business IDs (not numeric IDs)
- Typecheck must pass before committing: `pnpm typecheck`
