## Project Overview

echoe is a full-stack AI-powered note-taking and knowledge management system. It's a pnpm monorepo with a React 19 frontend and Express.js backend using a dual-database architecture: MySQL (via Drizzle ORM) for relational data and LanceDB for vector search.
